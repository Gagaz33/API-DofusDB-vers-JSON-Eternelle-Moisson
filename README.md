# API-DofusDB-vers-JSON-Eternelle-Moisson
*Petit script Python qui depuis une liste JSON avec le nom des monstres, ajoute l'id_DofusDB, la famille, les zones et le donjon associé.*  
*Utilisé pour le site https://mopolo.dev/Eternelle-Moisson/*
  
### Placer le JSON d'origine dans le meme dossier sous le nom "monsters_originel.json" et executer le script.
